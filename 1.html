HTML

Hypertext Markup Language (HTML).HTML provides a way to create structured documents with links, citations, and other items, as well as expressing structural meaning for text such as headings, paragraphs, and lists . And it can be used to embed images and objects and create interactive forms. HTML is written in the form of HTML elements that are "tags" enclosed in angle brackets within the web page content .

HTML is a Web browser that affect the behavior of HTML processing device, such as a JavaScript that defines the appearance and layout of text and other items CSS same script can contain or recall. The W3C , co-director of the HTML and CSS standards , recommends the use of CSS for clear and representative markup.
